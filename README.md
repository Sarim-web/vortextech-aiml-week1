# 📊 Vortex Tech AI/ML Internship – Week 1

## Data Cleaning & Exploratory Data Analysis (EDA)

This repository contains my Week 1 assignment for the **Vortex Tech AI & Machine Learning Internship Program**. The objective of this project was to perform data cleaning, exploratory data analysis (EDA), and basic visualization using Python and the Pandas library.

---

## 📌 Project Overview

In every AI and Machine Learning project, understanding and preparing the dataset is a critical first step. This project demonstrates the complete workflow of:

* Loading a real-world dataset
* Exploring the dataset structure
* Identifying data quality issues
* Cleaning and preparing the data
* Generating summary statistics
* Visualizing important trends
* Documenting observations

The project uses the **IBM HR Employee Attrition Dataset**, a widely used dataset for practicing data analysis and machine learning concepts.

---

## 🎯 Objectives

* Load and inspect a real-world dataset
* Identify missing values, duplicates, and incorrect data types
* Clean and preprocess the data
* Generate descriptive statistics
* Create meaningful visualizations
* Document findings using a Jupyter Notebook

## 📈 Data Cleaning Process

The following preprocessing steps were performed:

* Loaded the dataset using Pandas.
* Inspected the dataset structure.
* Checked for missing values.
* Checked for duplicate records.
* Verified column data types.
* Removed columns containing only a single unique value (no analytical value).
* Saved the cleaned dataset for future use.

---

## 📊 Exploratory Data Analysis

The notebook includes:

* Dataset overview
* Summary statistics
* Employee attrition analysis
* Department-wise attrition visualization
* Employee age distribution
* Category frequency analysis

---

## 🔍 Key Findings

* The dataset contains **1,470 employee records**.
* No missing values were found.
* No duplicate records were detected.
* Constant-value columns were removed as they provided no useful information for analysis.
* Employee attrition trends differ across departments.
* The age distribution provides insight into the workforce demographics.

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/Sarim-web/vortextech-aiml-week1.git
```

2. Navigate to the project directory.

```bash
cd vortextech-aiml-week1
```

3. Install the required libraries.

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook.

```bash
jupyter notebook
```

5. Open the notebook inside the `notebooks` folder and run all cells.
