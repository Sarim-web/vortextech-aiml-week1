# Vortex Tech AI/ML Internship - Week 1

This project contains the Week 1 HR employee attrition data cleaning workflow.

## Structure
- data/raw/WA_Fn-UseC_-HR-Employee-Attrition.csv
- notebooks/week1_data_cleaning.ipynb
- README.md
- requirements.txt

## Setup
Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage
1. Place the dataset in the data/raw folder.
2. Open the notebook in the notebooks folder.
3. Run the cells from top to bottom.
